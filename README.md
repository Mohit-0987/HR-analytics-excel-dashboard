# 📊 HR Analytics Dashboard (Excel)

## 📌 Project Overview
This project is an interactive **HR Analytics Dashboard** built using Microsoft Excel.  
It analyzes employee attrition patterns and provides key business insights using pivot tables, charts, and slicers.

---

## 🎯 Objective
To understand:
- Why employees are leaving
- Which departments are most affected
- How factors like age, salary, satisfaction, and tenure impact attrition

---

## 📁 Dataset
- Source: IBM HR Analytics Dataset (Kaggle)
- File: `IBM HR Analytics Employee Attrition & Performance`

---

## 🛠️ Tools Used
- Microsoft Excel
  - Pivot Tables
  - Pivot Charts
  - Slicers
  - Excel Formulas

---

## 📊 Dashboard Features

### 🔹 KPI Metrics
- Total Employees: **1470**
- Total Attrition: **237**
- Attrition Rate: **16.1%**
- Average Salary: **6500**
- Average Tenure: **7 years**

---

### 🔹 Visual Analysis
- Attrition by Department
- Attrition by Age Group
- Attrition by Job Satisfaction
- Attrition by Gender
- Average Salary by Attrition
- Attrition by Tenure

---

### 🔹 Interactivity
- Slicers used:
  - Department
  - Gender
  - Job Role

Users can dynamically filter the dashboard to explore different scenarios.

---

## 📈 Key Insights
- Highest attrition observed in **Research & Development department**
- Employees aged **26–35** show higher attrition
- Mid-level job satisfaction (level 3) has unexpectedly high attrition
- Employees with **lower salaries** are more likely to leave
- **Early tenure employees** (0–2 years) have higher attrition
